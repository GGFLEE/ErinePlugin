##  配置

1. 访问[文心一言](https://yiyan.baidu.com/).
2. 打开开发者工具.
3. 找到应用程序(Application).
4. 在左侧点击存储(Storage)-Cookies-[https://yiyan.baidu.com](https://yiyan.baidu.com/).
5. 在列表中点击BAIDUID.
6. 复制下方Cookie Value的值.
7. BDUSS_BFESS同理.
8. 控制台填写配置
